不适合日用。 Camoufox 本质上是一个为自动化/爬虫场景定制的 Firefox 改版，不是给普通人当日常浏览器用的。



为什么不适合


1.

定位就不是日用浏览器

官方定义是 "built for AI agents"，面向 Playwright 自动化场景
核心卖点是指纹伪装、反检测——这些对日常上网毫无意义，反而添麻烦
2.

底层改动会破坏正常体验

指纹注入/轮换意味着你每次访问同一个网站，网站看到的可能是不同的设备、分辨率、字体集
WebGL、Canvas 都被篡改过，可能导致部分网页渲染异常
WebRTC IP 伪装可能影响视频通话、WebRTC 类应用
3.

插件生态不友好

虽然基于 Firefox，但不是标准 Firefox 构建，扩展兼容性没有保障
没有同步功能、没有账号体系
4.

稳定性警告

官方明确说了：2026 新版本是 "highly experimental"，有 breaking changes，不适合生产环境
原作者已退位，新团队（Clover Labs）还在追赶中
5.

性能配置

虽然比 Chrome 轻（~200MB vs 800MB+），但那是精简掉了正常浏览器该有的东西换来的


总结


场景	适合？
WebAI2API 这类自动化桥接	✅ 完美
爬虫 / 数据采集	✅ 很强
日常刷网页、看视频、办公	❌ 别折腾
开发调试	⚠️ 用标准 Firefox/Chrome DevTools 更好



日用老老实实用 Firefox / Chrome / Arc 这类正经浏览器就好，别把工具当玩具 😄

能，而且不需要劫持系统对话框。Camoufox 基于 Playwright，Playwright 的 setInputFiles 直接绕过原生文件选择框，完全在浏览器层面搞定。



原理



原生文件对话框（Windows 资源管理器 / macOS Finder）是操作系统级的，浏览器自动化工具控制不了它。但正确做法不是去劫持它，而是根本不让它弹出来：



text
text
用户点击 <input type="file"> → 弹出系统对话框 → 用户选文件 → 浏览器拿到文件

自动化路径：
setInputFiles() → 直接把文件塞进 input 元素 → 对话框根本不出现



Camoufox（Playwright）的方案


python
python
# Python 示例 - Camoufox 用法和 Playwright 一样
from camoufox.sync_api import Camoufox

with Camoufox(headless=False) as browser:
    page = browser.new_page()
    page.goto("https://example.com/upload")

    # 方法1：直接指定文件路径（最常用）
    page.set_input_files("input[type='file']", "/path/to/file.png")

    # 方法2：多个文件
    page.set_input_files("input[type='file']", [
        "/path/to/file1.png",
        "/path/to/file2.jpg",
    ])

    # 方法3：不用真实文件，用内存数据
    page.set_input_files("input[type='file']", {
        "name": "test.txt",
        "mimeType": "text/plain",
        "buffer": b"file content here"
    })

    # 方法4：拖拽上传（drag and drop 区域）
    # 先找到 drop zone，再 setInputFiles
    drop_zone = page.locator(".drop-zone")
    drop_zone.set_input_files("/path/to/file.png")



如果非要处理原生对话框（不推荐）



有些场景 <input> 元素被隐藏或框架没暴露标准 input，这时候才需要兜底方案：



python
python
# 方案A：监听 file chooser 事件
with page.expect_file_chooser() as fc_info:
    page.click("#upload-button")  # 触发弹窗
file_chooser = fc_info.value
file_chooser.set_files("/path/to/file.png")  # Playwright 拦截了对话框

# 方案B：强制让隐藏的 input 可见再操作
page.evaluate("""
    document.querySelector('input[type="file"]').style.display = 'block'
""")
page.set_input_files("input[type="file"]", "/path/to/file.png")



各种上传方式对照


方式	需要对话框？	可靠性	适用场景
setInputFiles	❌ 不弹	⭐⭐⭐⭐⭐	标准 <input type="file">
expect_file_chooser	❌ 拦截	⭐⭐⭐⭐	按钮触发的上传
evaluate + setInputFiles	❌ 不弹	⭐⭐⭐⭐	隐藏 input 元素
xdotool / pyautogui	✅ 需要	⭐⭐	真的要操作系统对话框
AutoHotKey (Win)	✅ 需要	⭐⭐	Windows 原生对话框


结论



99% 的情况不需要劫持系统对话框。 Playwright 的 setInputFiles 和 expect_file_chooser 能覆盖绝大多数上传场景，Camoufox 完全继承了这套能力。只有在极端情况下（比如某些 Electron 应用的自定义文件选择器）才需要考虑 xdotool / pyautogui 这种系统级方案。

class MessageBus:
    """
    AI 之间所有交互都经过这里
    - 可以注入（手动插入指令）
    - 可以拦截（发现问题暂停）
    - 可以重放（回溯整个过程）
    """

    def __init__(self):
        self.channels: dict[str, list[Message]] = {}  # 按工作流分通道
        self.hooks: list[Callable] = []  # 中间件：日志、过滤、通知

    async def send(self, from_id: str, to_id: str, content: str, metadata: dict):
        """发送消息给某个 AI 实例"""
        msg = Message(from_id, to_id, content, metadata)

        # 中间件处理
        for hook in self.hooks:
            msg = await hook(msg)
            if msg is None:
                return  # 被拦截了

        # 把内容注入到目标实例的浏览器输入框
        await self.inject_to_browser(to_id, msg)

    async def receive(self, from_id: str) -> str:
        """从某个 AI 实例收集输出"""
        # 监听浏览器中 AI 的回复区域
        response = await self.collect_from_browser(from_id)

        # 触发钩子（记录日志、检查内容等）
        for hook in self.hooks:
            await hook(Message(from_id, None, response))

        return response

    async def inject_to_browser(self, instance_id: str, msg: Message):
        """
        把上一个 AI 的输出「喂」给下一个 AI
        这里是核心：模拟人工粘贴内容到输入框

        page = self.get_page(instance_id)

        # 找到输入框，注入内容
        input_box = page.locator("textarea, [contenteditable]")

class WorkflowEngine:
    def __init__(self, bus: MessageBus):
        self.bus = bus

    async def run_pipeline(self, prompt: str, steps: list[PipelineStep]):
        """流水线模式"""
        current_content = prompt
        results = []

        for step in steps:
            # 发给指定 AI
            await self.bus.send("orchestrator", step.instance_id, current_content)
            # 等它回复
            response = await self.bus.receive(step.instance_id)
            results.append({"step": step.name, "output": response})
            # 回复作为下一步的输入
            current_content = response

        return results

    async def run_roundtable(self, topic: str, participants: list[str], rounds: int = 3):
        """圆桌讨论模式"""
        discussion_log = []
        current_statement = topic

        for i in range(rounds):
            for participant in participants:
                # 把之前的讨论历史 + 当前话题喂给这个 AI
                context = self._build_context(discussion_log, topic)
                await self.bus.send("orchestrator", participant, context)
                response = await self.bus.receive(participant)

                discussion_log.append({
                    "round": i + 1,
                    "speaker": participant,
                    "content": response
                })

                # 关键：每个 AI 都能看到前面所有人的发言
                # 所以它能真正「回应」而不是自说自话

        return discussion_log

    async def run_review_loop(self, coder: str, reviewer: str, task: str, max_rounds: int = 5):
        """审核循环模式"""
        await self.bus.send("orchestrator", coder, task)
        code = await self.bus.receive(coder)

        for i in range(max_rounds):
            review_prompt = f"请审查以下内容，如果有问题请指出并给出修改建议，如果没问题请说「通过」：\n\n{code}"
            await self.bus.send("orchestrator", reviewer, review_prompt)
            review = await self.bus.receive(reviewer)

            if "通过" in review:  # 简化判断
                return {"final": code, "review": review, "rounds": i + 1}

            # 没通过，把 review 意见喂回去让 coder 改
            await self.bus.send("orchestrator", coder, f"审查意见如下，请修改：\n\n{review}")
            code = await self.bus.receive(coder)

        return {"final": code, "note": "达到最大轮次", "rounds": max_rounds}

    async def run_debate(self, topic: str, affirmative: str, negative: str, judge: str, rounds: int = 3):
        """辩论模式"""
        debate_log = []

        for i in range(rounds):
            # 正方发言（能看到之前所有内容）
            context = self._build_context(debate_log, f"辩题：{topic}\n你是正方，请发言：")
            await self.bus.send("orchestrator", affirmative, context)
            pos = await self.bus.receive(affirmative)
            debate_log.append({"round": i+1, "side": "正方", "content": pos})

            # 反方反驳
            context = self._build_context(debate_log, f"辩题：{topic}\n你是反方，请反驳：")
            await self.bus.send("orchestrator", negative, context)
            neg = await self.bus.receive(negative)
            debate_log.append({"round": i+1, "side": "反方", "content": neg})

        # 评委总结
        full_debate = self._build_context(debate_log, "请作为评委总结双方论点并给出评判：")
        await self.bus.send("orchestrator", judge, full_debate)
        verdict = await self.bus.receive(judge)

        return {"debate": debate_log, "verdict": verdict}

# adapters/base.py
from abc import ABC, abstractmethod
from playwright.async_api import Page

class BaseAdapter(ABC):
    """所有平台适配器的基类"""

    # ── 平台元信息 ──
    platform_name: str          # "chatgpt"
    platform_url: str           # "https://chatgpt.com"
    needs_login_check: bool     # 是否需要登录态检测

    # ── DOM 选择器（每个平台不同，写在子类里）──
    selector_input: str         # 输入框选择器
    selector_send_btn: str      # 发送按钮（有的平台没有，用 Enter）
    selector_response: str      # 回复内容区域
    selector_stop_btn: str      # "停止生成"按钮
    selector_login_form: str    # 登录表单（判断是否掉线）
    selector_captcha: str       # 验证码元素
    selector_new_chat: str      # 新建对话按钮
    selector_quota_info: str    # 额度信息（如果有的话）

    def __init__(self, page: Page):
        self.page = page

    # ── 生命周期 ──
    @abstractmethod
    async def init(self) -> None:
        """打开页面、等待加载完成"""

    @abstractmethod
    async def ensure_logged_in(self) -> bool:
        """确认已登录，返回 True/False"""

    # ── 核心交互 ──
    @abstractmethod
    async def send_message(self, text: str) -> None:
        """在输入框输入文本并发送"""

    @abstractmethod
    async def collect_response(self, on_chunk: Callable[[str], None]) -> str:
        """
        流式收集 AI 回复
        on_chunk: 每收到一段文字回调一次（用于实时推送到前端）
        返回完整回复
        """

    @abstractmethod
    async def new_conversation(self) -> None:
        """开一个新对话"""

    # ── 状态检测 ──
    @abstractmethod
    async def check_status(self) -> InstanceStatus:
        """
        返回当前状态枚举：
        - ONLINE: 可用
        - LOGGED_OUT: 未登录/掉线
        - CAPTCHA: 需要人机验证
        - QUOTA_EXHAUSTED: 额度用完
        - RATE_LIMITED: 被限流
        - ERROR: 其他错误
        """

    @abstractmethod
    async def get_quota_info(self) -> QuotaInfo | None:
        """获取额度信息（剩余次数等），不支持返回 None"""

    # ── 截图 ──
    async def screenshot(self) -> bytes:
        """截图，子类可覆盖自定义区域"""
        return await self.page.screenshot(type="jpeg", quality=40)

# adapters/chatgpt.py
class ChatGPTAdapter(BaseAdapter):
    platform_name = "chatgpt"
    platform_url = "https://chatgpt.com"

    selector_input = "#prompt-textarea"
    selector_send_btn = 'button[data-testid="send-button"]'
    selector_response = '[data-message-author-role="assistant"]'
    selector_stop_btn = 'button[aria-label="Stop generating"]'
    selector_login_form = 'button[aria-label="Log in"]'
    selector_captcha = 'iframe[src*="recaptcha"]'
    selector_new_chat = 'a[href="/"]'

    async def init(self):
        await self.page.goto(self.platform_url, wait_until="domcontentloaded")
        await self.page.wait_for_load_state("networkidle")

    async def ensure_logged_in(self) -> bool:
        # 检查是否在对话界面（有输入框就说明已登录）
        try:
            await self.page.wait_for_selector(self.selector_input, timeout=10000)
            return True
        except:
            return False

    async def send_message(self, text: str):
        input_box = self.page.locator(self.selector_input)
        await input_box.fill(text)
        # ChatGPT 的输入框是 contenteditable，fill 之后还要触发一下
        await self.page.keyboard.press("Enter")

    async def collect_response(self, on_chunk=None) -> str:
        # 等回复区域出现
        await self.page.wait_for_selector(self.selector_response, timeout=30000)

        last_text = ""
        while True:
            # 读取最新一条回复的文本
            response_el = self.page.locator(self.selector_response).last
            current_text = await response_el.inner_text()

            if current_text != last_text:
                if on_chunk:
                    on_chunk(current_text[len(last_text):])
                last_text = current_text

            # 判断是否说完：停止生成按钮消失了
            stop_btn = self.page.locator(self.selector_stop_btn)
            if not await stop_btn.is_visible():
                # 再等一下确认没有新内容
                await asyncio.sleep(0.5)
                final_text = await response_el.inner_text()
                if final_text == last_text:
                    break
                last_text = final_text

            await asyncio.sleep(0.3)

        return last_text

    async def check_status(self) -> InstanceStatus:
        # 先看有没有验证码
        captcha = self.page.locator(self.selector_captcha)
        if await captcha.count() > 0:
            return InstanceStatus.CAPTCHA

        # 看有没有登录按钮
        login_btn = self.page.locator(self.selector_login_form)
        if await login_btn.count() > 0:
            return InstanceStatus.LOGGED_OUT

        # 看有没有输入框
        input_box = self.page.locator(self.selector_input)
        if await input_box.count() > 0:
            return InstanceStatus.ONLINE

        return InstanceStatus.ERROR

# workflow/bus.py
from dataclasses import dataclass, field
from typing import Callable, Awaitable
from enum import Enum

class MessageType(Enum):
    PROMPT = "prompt"           # 用户/上游发来的指令
    RESPONSE = "response"       # AI 的回复
    SYSTEM = "system"           # 系统消息（注入的上下文）
    CONTROL = "control"         # 控制信号（暂停/继续/终止）

@dataclass
class BusMessage:
    msg_type: MessageType
    from_id: str                # 发送者（实例ID 或 "user" 或 "orchestrator"）
    to_id: str                  # 接收者
    content: str                # 内容
    metadata: dict = field(default_factory=dict)
    # metadata 示例：
    # {"workflow_id": "wf-001", "step": 2, "round": 1, "side": "正方"}

class MessageBus:
    def __init__(self):
        self._handlers: dict[str, list[Callable[[BusMessage], Awaitable[None]]]] = {}
        self._history: list[BusMessage] = []  # 全量消息记录
        self._middleware: list[Callable[[BusMessage], Awaitable[BusMessage | None]]] = []

    def register(self, instance_id: str, handler: Callable):
        """注册一个实例的消息处理器"""
        self._handlers.setdefault(instance_id, []).append(handler)

    def use(self, middleware: Callable):
        """注册中间件（日志、过滤、注入等）"""
        self._middleware.append(middleware)

    async def send(self, msg: BusMessage):
        """发送消息"""
        # 过中间件
        for mw in self._middleware:
            msg = await mw(msg)
            if msg is None:
                return  # 被中间件拦截

        # 记录历史
        self._history.append(msg)

        # 投递到目标
        handlers = self._handlers.get(msg.to_id, [])
        for handler in handlers:
            await handler(msg)

    def get_history(self, workflow_id: str = None) -> list[BusMessage]:
        """获取消息历史，可按工作流过滤"""
        if workflow_id:
            return [m for m in self._history if m.metadata.get("workflow_id") == workflow_id]
        return self._history.copy()

    def clear_history(self, workflow_id: str = None):
        """清理历史"""
        if workflow_id:
            self._history = [m for m in self._history if m.metadata.get("workflow_id") != workflow_id]

    prompt_template: str        # 可引用上游输出："{prev.output}"

class WorkflowDef:
    id: str
    name: str
    mode: str                   # "pipeline" | "roundtable" | "review_loop" | "debate"
    steps: list[WorkflowStep]
    config: dict                # 模式专属配置

class WorkflowEngine:
    def __init__(self, bus: MessageBus, browser_pool):
        self.bus = bus
        self.pool = browser_pool
        self.active_workflows: dict[str, WorkflowRunner] = {}

    async def execute(self, wf: WorkflowDef, initial_input: str) -> str:
        runner = WorkflowRunner(wf, self.bus, self.pool)
        self.active_workflows[wf.id] = runner
        try:
            return await runner.run(initial_input)
        finally:
            del self.active_workflows[wf.id]

    async def pause(self, wf_id: str):
        """暂停工作流"""
        if runner := self.active_workflows.get(wf_id):
            await runner.pause()

    async def resume(self, wf_id: str):
        """恢复工作流"""
        if runner := self.active_workflows.get(wf_id):
            await runner.resume()

    async def abort(self, wf_id: str):
        """终止工作流"""
        if runner := self.active_workflows.get(wf_id):
            await runner.abort()

    async def inject(self, wf_id: str, step_id: str, content: str):
        """
        人工注入：在某个步骤执行前插入额外指令
        比如：AI 正在讨论，你突然想加一个观点
        """
        if runner := self.active_workflows.get(wf_id):
            await runner.inject(step_id, content)
