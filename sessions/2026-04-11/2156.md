1. 🦖 Playwright 自带 Chromium（最大杀手）



extraResources 把 playwright/.local-browsers 下的完整 Chromium 打进去了，一个浏览器 ~200MB。




方案：让用户已有的浏览器，或下载独立轻量 Chromium

2. 📦 ASAR 压缩没开



asar: true 只是打包，没压缩。加上：



json
json
"asar": true,
"asarUnpack": ["**/*.node", "**/better-sqlite3/**"],
"compression": "maximum"




compression: "maximum" 让 electron-builder 用最大压缩率打包。





3. 🧹 排除不需要的文件



当前 files 排除规则太松。加强：



json

"files": [

  "!**/node_modules/**/*.d.ts",
  "!**/node_modules/.bin/**",
  "!**/node_modules/**/README*",
  "!**/node_modules/**/CHANGELOG*",
  "!**/node_modules/**/LICENSE*",
  "!**/node_modules/**/{example,examples,demo,demos}/**",
  "!**/node_modules/**/{docs,doc}/**",
  "!**/node_modules/**/*.map",
  "!**/node_modules/**/*.ts",
  "!**/node_modules/**/test.*"

最根本的方案——不要每次发全量包，用 differential update：



json

js
js
// 之前（报错）
import { autoUpdater } from 'electron-updater'

// 之后（正确）
import pkg from 'electron-updater'
const { autoUpdater } = pkg
