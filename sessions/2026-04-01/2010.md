GX Works2 导入指令表 (.il) 的方法



GX Works2 对 .il 文件（助记符/指令表）的支持有限，实际使用中有以下几种路径：





方法一：直接粘贴指令表（推荐，最实用）


1.
打开 GX Works2 → 新建工程 → 选择你的 PLC 系列（如 FX 系列）
2.
切换到 梯形图编辑模式（编辑 → 梯形图）
3.
菜单 → 编辑 → 文本模式切换（或按 Alt + X）
4.
在文本编辑框中粘贴指令表内容
5.
再切回梯形图模式（Alt + X），软件自动将指令表转为梯形图
6.
检查 → 编译 → 下载到 PLC



⚠️ 粘贴时注意格式：GXW2 的助记符格式是 LD X0、OUT Y0 这种，工具导出的 .il 基本兼容。如果有中文注释，先删掉注释行再粘贴。





方法二：导入文本文件


1.
菜单 → 工程 → 读取 → 其他数据格式
2.
选择 "文本文件(*.txt)"
3.
选择导出的 .il 文件
4.
软件尝试解析并转为梯形图
5.
检查解析结果，手动修正



这个功能在不同版本的 GXW2 上行为有差异，老版本可能不支持。





方法三：先用 GX Developer 做中转



GX Developer（GXW2 的前身）对文本指令表的兼容性更好：



1.
GX Developer → 新建工程 → 选择 PLC 类型
2.
菜单 → 工程 → 读取 → 选择文本文件
3.
导入成功后保存为 .gxw 工程
4.
用 GX Works2 打开 .gxw 文件 → 自动升级转换
